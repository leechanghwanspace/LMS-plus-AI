# RedPenLMS
<br>

**LMS💻 + AI🤖, learning assistant platform**<br>

---
<br>

### **_플랫폼 기능_**


- 소프트웨어학부 전용 LMS  + AI **설계**
  - 학생의 희망 진로/직무 **선택**
  - 직무 별 교과목 강의 **추천**
  - 교과목 관련 인프런 강의 **추천**
- 주차별 학습 계획 및 성과 **확인**
--- 
### **추가 예정 기능**
- 교과목에 대한 강의평(후기) **제공**
- 전공도서 **중고거래**
